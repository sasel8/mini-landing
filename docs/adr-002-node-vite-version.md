# ADR-002: Node ve Vite Sürüm Uyumu

- Karar: Node 18 kullanmaya devam edebilmek için Vite 7 yerine Vite 5 kullanılması.
- Neden: Mevcut ortamda Node sürümünü yükseltmek yerine proje bağımlılığını aşağı çekmek daha hızlı ve risksizdi.
- Sonuç: Vite 5 ile proje stabil şekilde çalıştı ve geliştirme sorunsuz ilerledi.
